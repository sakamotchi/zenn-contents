# はじめに

SQL構文を覚えるのが面倒、本番環境でうっかり危険な操作をしてしまいそう...そんな悩みを解決するため、GUI操作で直感的にSQLクエリを構築できるデスクトップアプリケーション「SQL Query Builder」を作りました。

https://github.com/sakamotchi/sql-query-build

**このアプリケーションのコードは、すべてAIコーディングエージェントを活用して実装しました。** 具体的には、Claude Code、ChatGPTのCodex CLI、Gemini（Antigravity）を駆使して開発を進めました。AIの力を借りることで、複雑なRustコードやVue.jsのコンポーネント設計を効率的に実装できました。

# 主な特徴

## 1. GUIによるクエリ構築

テーブルやカラムをGUIで選択するだけで、バックグラウンドで自動的にSQLが生成されます。SQL構文の知識がなくても、直感的な操作でクエリを作成できます。複雑なJOINやGROUP BYも、フォームに入力するだけで正確なSQLを生成します。

## 2. セキュリティ重視の設計

本番環境への誤操作を防ぐため、以下の機能を実装しています：

- **環境別ウィンドウカラーリング**: 本番環境、ステージング環境、開発環境でウィンドウの色を変えることで、視覚的に環境を識別
- **危険操作の検出**: WHERE句なしのDELETEなど、危険な操作を検出して警告
- **AES-256-GCM暗号化**: データベース接続情報のパスワードを暗号化して安全に保存

## 3. マルチデータベース対応

以下の3つのデータベースに対応しています：

- PostgreSQL
- MySQL
- SQLite

複数の接続を同時に管理でき、環境ごとに切り替えて使用できます。

## 4. 充実した機能

- SELECT、INSERT、UPDATE、DELETEの構築
- JOIN（INNER JOIN、LEFT JOIN、RIGHT JOIN、FULL OUTER JOIN）
- GROUP BY、ORDER BY
- CSV形式でのエクスポート
- クエリ履歴と保存機能

## 開発手法

1. **開発速度の向上**: 通常なら数週間かかる実装を短期間で完成
2. **品質の向上**: AIが提案するベストプラクティスを採用
3. **学習効果**: 不慣れなRustやTauriの学習を効率化
4. **コードレビュー**: AIが潜在的なバグやセキュリティ問題を指摘

# なぜ作ったのか

データベース操作を行う際、以下のような課題を感じていました：

1. **SQL構文の記憶負担**: JOIN構文やGROUP BYの細かい書き方を毎回調べるのが手間
2. **本番環境での誤操作リスク**: コマンドラインやSQLクライアントで本番環境を操作する際の緊張感
3. **複雑なクエリの可視化**: 複雑なJOINやサブクエリを視覚的に理解したい

これらの課題を解決するため、GUIベースでセキュアなクエリビルダーを開発しました。

SQL Query Builderは、GUI操作でSQLクエリを構築できるデスクトップアプリケーションです。セキュリティを重視した設計により、本番環境での誤操作を防ぎながら、効率的にデータベース操作を行えます。
